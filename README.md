# Curso EIE 390  
## Robótica e Inteligencia Artificial  

### Guías de Laboratorio  

Este repositorio contiene las **guías de laboratorio** del curso **EIE 390: Robótica e Inteligencia Artificial**, orientadas al aprendizaje práctico de conceptos fundamentales de robótica, programación y sistemas inteligentes.

Las guías están diseñadas para acompañar el desarrollo teórico del curso, abordando temas como:
- Programación en Python y MicroPython
- Sensores y actuadores
- Sistemas embebidos
- Robótica móvil
- Introducción a la inteligencia artificial aplicada a robótica

Cada guía incluye objetivos, desarrollo paso a paso y ejercicios prácticos para reforzar los contenidos vistos en clases.
